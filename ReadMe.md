# MoveBlocks

 - A Hypixel-Like Cosmetic Plugin
 - Fully Configurable
 - Uses Inventory Menu

# Configuration
```
 - The how many times it updates in ticks
update-time: 4
 - The worlds that the plugin doesnt run in
banned-worlds:
- "world1"
- "world2"
 - The blocks that will be accessible in the menu
blocks:
"1:0":
permission: "mb.stone"
```
